- BeforeRouter: filter is executed before the request reaches the router, 
allowing you to perform authentication, authorization, or other checks. 
It's typically used for tasks like logging, authentication, or setting up session data 
- BeforeExec: This filter is executed before the execution of the controller 
method, allowing you to perform tasks like setting up the request or response 
objects
- AfterExec: This filter is executed after the execution of the controller 
method, allowing you to perform tasks like logging or setting up the response 
- FinishRouter: This filter is executed after the request has been processed 
by the router, allowing you to perform tasks like setting up the response or 
logging
